# 集成 CollectionManager 与 FloatingActionButton 计划

## 1. 修改 HeaderContext.tsx
- 添加 `isCollectionManagerOpen` 和 `setIsCollectionManagerOpen` 状态
- 用于控制 CollectionManager 组件的显示/隐藏

## 2. 修改 Header.tsx
- 导入 `CollectionManager` 组件
- 在组件底部添加 `CollectionManager` 组件的渲染
- 传递必要的 props，包括状态和回调函数

## 3. 修改 FloatingActionButton.tsx
- 将默认操作配置中的"添加收藏"和"修改收藏"按钮替换为一个统一的"添加/修改收藏"按钮
- 修改 `handleActionClick` 函数，处理统一按钮的点击事件，打开 CollectionManager
- 移除不再使用的 `handleAddCollection` 和 `handleEditCollection` 函数

## 4. 评估代码实现
- 比较 `CollectionEditForm` 和 `CollectionManager` 的功能
- 决定保留 `CollectionEditForm` 或移除它，将其功能合并到 `CollectionManager` 中

## 5. 测试集成
- 确保点击 FloatingActionButton 中的统一按钮能正确打开 CollectionManager
- 确保 CollectionManager 能正常工作，包括搜索、添加和修改收藏功能
- 确保操作成功后能正确触发回调函数

## 6. 代码优化
- 移除不再使用的代码
- 确保代码结构清晰，易于维护
- 确保所有组件都能正确协同工作
