# 问题一：添加条目信息功能中sid参数处理方案

## 业务逻辑设计
1. **当用户未提供sid时**：
   - 自动生成新的subject ID
   - 完整创建新条目和收藏记录
   - 返回新创建的subject ID给前端

2. **具体实现**：
   - 修改`upsert_collection_endpoint`函数，将sid参数改为可选
   - 添加逻辑判断：如果sid为None，生成新条目ID
   - 确保生成的ID不与现有ID冲突
   - 返回包含新ID的完整响应

## 错误处理机制
1. **缺少必要字段时**：返回400 Bad Request，明确错误信息
2. **数据库操作失败时**：返回500 Internal Server Error，记录详细日志
3. **无效数据格式时**：返回400 Bad Request，包含具体验证错误

## 默认行为
1. **自动生成sid**：使用数据库自增ID
2. **返回完整信息**：包括新生成的subject ID和收藏详情
3. **保持API一致性**：返回格式与原有API相同

# 问题二：收藏表单设计与API调用方案

## 表单字段设计

### 核心收藏字段
1. **收藏状态**：下拉选择（1=想看/2=看过/3=在看/4=搁置/5=抛弃）
2. **评分**：滑块（0-10，0表示未评分）
3. **评论**：多行文本输入
4. **是否私有**：复选框
5. **标签**：输入框，支持逗号分隔

### 新条目字段（仅创建新收藏时显示）
1. **条目名称**：文本输入
2. **中文名称**：文本输入
3. **条目类型**：下拉选择（1=书籍/2=动画/3=音乐/4=游戏/6=三次元）
4. **封面URL**：文本输入
5. **上映日期**：日期选择器（YYYY-MM-DD）
6. **发售日期**：日期选择器（YYYY-MM-DD）

## 数据验证规则
1. **必填字段验证**：收藏状态、条目名称、类型、封面URL
2. **日期格式验证**：YYYY-MM-DD格式
3. **日期逻辑验证**：上映日期 ≤ 发售日期
4. **评分范围验证**：0-10整数
5. **标签格式验证**：逗号分隔的字符串列表
6. **最大长度验证**：评论（2000字符）、标签（50字符/个）

## API调用参数格式
```json
{
  "collection": {
    "status": 2,
    "rate": 9,
    "comment": "很好看的动画",
    "private": false,
    "tags": ["动画", "热血"]
  },
  "subject": {
    "name": "测试动画",
    "name_cn": "测试动画中文名",
    "type": 2,
    "cover_url": "https://example.com/cover.jpg",
    "release_date": "2023-01-01",
    "publish_date": "2023-01-01",
    "tags": ["动画", "热血"]
  }
}
```

## 错误处理流程
1. **前端验证**：实时验证用户输入，显示即时错误提示
2. **API调用**：使用axios发送PUT请求到`/api/v1/collections/{sid}`
3. **错误捕获**：
   - 400错误：显示具体验证错误信息
   - 401错误：提示用户重新登录
   - 500错误：显示通用错误提示，记录详细日志
4. **成功处理**：关闭表单，刷新收藏列表，显示成功提示

# 问题三：按钮符号与字体居中对齐修复方案

## 问题分析
1. **CSS样式问题**：Button组件使用`inline-flex items-center justify-center gap-2`，但可能存在以下问题：
   - 不同大小的图标与文字基线对齐问题
   - 不同字体大小导致的垂直对齐问题
   - 行高(line-height)设置不当

## 解决方案
1. **统一行高**：为所有按钮尺寸设置合适的line-height
2. **垂直对齐修复**：为图标添加`align-middle`类
3. **调整gap值**：确保图标与文字间距一致
4. **修复truncate类影响**：为内容span添加适当的display属性

## 具体CSS修改
```css
/* 在Button组件的样式中添加 */
button > span {
  line-height: 1;
  display: inline-flex;
  align-items: center;
}

/* 确保图标垂直居中 */
button svg, button [aria-hidden="true"] {
  align-self: center;
  line-height: 1;
}
```

## 测试方案
