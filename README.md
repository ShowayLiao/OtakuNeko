# 🐱 OtakuNeko (御宅猫)

**OtakuNeko** 是一个基于 **Streamlit** 和 **LLM (大语言模型)** 构建的智能化私人番剧管理与分析助手。

它可以同步你的 Bangumi (bgm.tv) 收藏数据，利用 AI 深度分析你的看番口味，生成精美的年度总结海报，并提供个性化的番剧推荐。不仅仅是一个记录工具，更是你的二次元“赛博嘴替”。

  

## ✨ 主要功能

  * **📊 数据同步与管理**：一键拉取 Bangumi 收藏记录，自动整理“看过”、“在看”、“想看”列表。
  * **🧠 AI 深度画像**：
      * **成分分析**：通过 LLM 分析你的阅片历史，生成“纯爱战神”、“乐子人”等趣味标签。
      * **毒舌/专业模式**：支持切换“毒舌猫娘”或“专业评论家”语调，让数据分析不再枯燥。
  * **🏆 年度动画报告**：
      * 自动生成包含“年度声优”、“最忙月份”、“最佳动画”等 12 个维度的深度报告。
      * **自动绘图**：一键生成精美的年度总结海报 (4x3 网格布局)，支持下载分享。
  * **💡 智能推荐**：基于向量数据库 (Vector Store) 和 AI 理解，推荐真正符合你口味的冷门佳作或热门新番。
  * **🚀 极致便携**：内置 Windows 一键启动/配置脚本，无需懂代码，双击即可运行。

## 📂 目录结构


## 🛠️ 快速开始

### 1\. 环境准备

确保你的电脑已安装 [Python 3.8 或更高版本](https://www.python.org/downloads/)。

### 2\. 初始化配置

在项目根目录下，双击运行 **`一键配置.bat`**。

  * 脚本会自动创建虚拟环境 (`venv`)。
  * 自动安装所有必要的依赖库 (`requirements.txt`)。

### 3\. 配置 API Key

在项目根目录下创建一个名为 `.env` 的文件，并填入你的 DeepSeek API Key（用于驱动 AI 分析）：

```ini
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 4\. 启动程序

双击 **`无窗口启动.vbs`**。

  * 程序将在后台静默运行。
  * 默认浏览器会自动打开 `http://localhost:8501` 进入 WebUI。

> *如果需要查看报错信息进行调试，请运行 `启动程序.bat`。*

## 📖 使用指南

1.  **侧边栏控制台**：
      * 点击 **“🔄 一键全量更新”** 同步你的 Bangumi 数据。
      * 选择 **“助手风格”** (猫娘/评论家)。
      * 点击 **“🧩 扩展插件”** 中的按钮触发特定功能（如年度报告）。
2.  **对话交互**：
      * 在底部输入框输入指令，例如：“根据我最近看的番推荐几部”、“分析一下我的看番口味”。
3.  **关闭程序**：
      * 使用完毕后，点击侧边栏底部的 **“❌ 关闭程序”** 按钮即可安全退出。

## 📝 常见问题

**Q: 启动时提示中文乱码？**
A: `启动程序.bat` 已内置 UTF-8 编码修复，请确保直接双击该脚本运行。

**Q: 如何更新依赖？**
A: 如果添加了新功能，再次运行 `一键配置.bat` 即可更新依赖。

**Q: 图片加载失败？**
A: 请检查网络连接，程序需要访问 Bangumi 图片服务器。部分地区可能需要配置网络环境。
